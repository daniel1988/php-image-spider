## image-spider.php

[博客](http://dotphper.veralyelec.com/?p=162)

* 用法

```
$ php image-spider.php -u http://www.meizitu.com/ -d /data/imgs
```

* 参数说明

    -u 网站链接

    -d 保存图片路径
